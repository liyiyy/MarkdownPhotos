## markdown 个人博客
喜欢MarkDown的书写方式，简单而一致，但遗憾的是不同的软件都markdown的支持都不同，github上的就支持的很简单了,真的好遗憾啊！
平时不喜欢写博客，文字表达能力不好，字不圆腔，总觉得少了点什么，但是有些东西不记录下来容易忘，不分享出来就不能和大家一起交流，不能获得更多，道理是明白，虽然懒但不能不做，有空的时候还是要写点啥。还有，写博客真的很费时间，但很值，写的过程中会不断地思考。
